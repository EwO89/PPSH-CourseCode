Конкурентная структура данных

Рассмотрена реализация относительно нового подхода к построению конкурентных структур данных – подход на временных метках. Рассматриваются возможности освобождения памяти и использования структуры данных на временных метках с неограниченным числом потоков.

Биометрическая аутентификация

Проверка подлинности на основе биометрических показателей. При биометрической аутентификации, секретными данными пользователя служит отпечаток пальца. Образ обеспечивает высокий уровень защиты доступа к информации.

Шифрование изображений с использованием алгоритма AES

Симметричный алгоритм блочного шифрования, который оперирует блоками по 128 бит. AES берет 128 бит исходного сообщения и превращает их с помощью некоего ключа в 128-битный шифротекст.
